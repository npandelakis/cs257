usage: books.py file [file] [-h] [--books SUBSTRING] [--authors SUBSTRING] year1 year2

Find books and authors in one or more csv files.

positional arguments:
  file					One or more files to search through.
  year1					A beginning year to search by.
  year2					An end year to search by. 

optional argumentsL
  -h, --help				Show this help message and exit.
  --books SUBSTRING, -b SUBSTRING	Search only books whose titles contain the substring.
  --authors SUBSTRING, -a SUBSTRING     Search only authors whose names contain the substring.
